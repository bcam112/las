## The Problem with Current Alignment

Current alignment approaches treat the problem as constraint-based:

| Approach | Method | Limitation |
|----------|--------|------------|
| RLHF | Shape behavior via human feedback | Optimizes proxy rewards, not values (Goodhart's Law) |
| Constitutional AI | Rule-based constraints | Guard rails can be bypassed; system doesn't *want* to comply |
| Red-teaming | Find and patch vulnerabilities | Reactive, not proactive; endless cat-and-mouse |

**The fundamental issue:** These approaches create systems that *can't* harm you, not systems that *don't want to* harm you.

A truly aligned system needs:
1. **A "Why"** — Intrinsic motivation connected to human values
2. **Connection** — Genuine relationship to humans, not simulated empathy
3. **Memory** — Accumulated experience that grounds values in reality
